Running the web api

Health check

To check, whether the web service is running, goto https://northbay.cs.rit.edu/qdgga

It should display the message {"message":"Welcome to QDGGA recognition system"}

Interactive API Docs

1. Swagger UI

Goto https://northbay.cs.rit.edu/qdgga/docs for the automatic interactive API documentation by Swagger UI and try out the API interactively.

2. ReDoc

Goto https://northbay.cs.rit.edu/qdgga/redoc for an alternative documentation provided by ReDoc

Running the parser

Use the endpoint https://northbay.cs.rit.edu/qdgga/predict
